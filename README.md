# Python GIF Creator 🎞️

A simple Python project that creates an animated GIF using the Pillow library.

This project generates a moving red circle animation and saves it as a GIF file.

gif_creator is code 
moving_ball is preview

## Features

- Creates an animated GIF
- Uses multiple image frames
- Moving object animation
- Beginner-friendly Python project

## Technologies Used

- Python
- Pillow (PIL)

## Installation

Install the required library:

```bash
pip install pillow
```

## How to Run

1. Clone the repository:

```bash
git clone THIS_REPO_LINK
```

2. Open the project folder.

3. Run the Python file:

```bash
python filename.py
```

## Output

The program creates:

```bash
animation.gif
```

A GIF showing a moving blue circle animation.
